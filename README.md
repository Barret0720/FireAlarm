# 一個住警器 守護一個家

## 作品緣由
近年來各地消防局力推住警器安裝政策，但是還是有許多民眾不了解住警器的功用與使用方法。COVID-19 疫情期間，分隊的防火宣導學姊們無法到民眾家中做居家訪視宣導，因此想透過這個網站，讓更多民眾知道住警器的功用與成功守護的案例。
